How the masking “safety value” is calculated

🏥 This advisory guides masking for hospitals & healthcare facilities — not a public/region-wide mandate.

The masking advisory does not come from the wastewater sample data. It is computed from real ICMR lab-confirmed respiratory-virus positivity (national), comparing the current week against its own recent baseline. Below is the live data and the exact formula — nothing hidden.

The formula

  1. Take ICMR positivity — weekly lab-confirmed counts for each respiratory pathogen (H1N1, H3N2, Influenza B, SARS-CoV-2, …).
  2. Sum per week → a single “respiratory positivity” total for each week.
  3. Current (X) = the latest week’s total. Baseline (Y) = the median of the previous 12 weeks.
  4. Ratio = X ÷ Y. Map it to a level: < 1.1 → not required 1.1–1.5 → suggested ≥ 1.5 → strongly advised

Using a ratio (current vs the area’s own recent baseline) is robust to scale — it flags a real rise in respiratory activity rather than an absolute number. Thresholds are a config and can be tuned with public-health input.
